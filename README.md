Dario
===========================

Fast and furious tile- and featureserver

Features:

- written in C#
- based on Web Api
- running on OWIN
- 100% OGC free

Api:
- http get /api: returns homedocumet
- http get /api/version: returns version
- http get /api/tile: returns very first tile 

Roadmap:

- PostGIS support
- Tileservices support
- GeoJSON support
- and many more :-)
